# Dark Facts Media Website

This is a static website designed for free hosting on Cloudflare Pages or GitHub Pages.

## Before publishing

Use GitHub's search-and-replace feature, or edit the files manually, to replace:

- `https://yourdomain.com` with your real domain
- `contact@yourdomain.com`
- `business@yourdomain.com`
- `privacy@yourdomain.com`
- `copyright@yourdomain.com`
- the `#` social links in the footer with your actual channel URLs

## Deploy with Cloudflare Pages

1. Create a GitHub repository.
2. Upload every file and folder from this package.
3. In Cloudflare, open Workers & Pages.
4. Create a Pages project and connect the GitHub repository.
5. Framework preset: None.
6. Build command: leave blank.
7. Build output directory: `/`
8. Deploy.
9. Add your domain under Custom Domains.

## Editing

Edit any `.html` file in GitHub and commit the change. Cloudflare will redeploy automatically.
